== Description ==
RomethemeForm For Elementor Plugin is an Form Builder for Elementor, and Widget Ready to use.

Features
• Form Builder
• Entries View

Form Widget
• Text Field
• Email Field
• Text Area Field
• Date Field
• Time Field
• Submit Button
